# \# Inventory Checker# # Compares inventory across Shopee, Lazada, and Zalora against Ordazzle and SAP, then outputs a colour-coded Excel report.# # \---# # \## Setup# # ```bash# pip install openpyxl requests# ```# # \### Credentials (required for Zalora API)# # ```bash# \# in inventory\_pkg/# cp credentials.example.py credentials.py# ```# # Then open `credentials.py` and fill in your Zalora `client\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\_id` and `client\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\_secret` per brand. This file is gitignored and will never be pushed to GitHub.# # \---# # \## Running# # `cd` into the folder that \*\*contains\*\* `inventory\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\_pkg/`, then:# # ```bash# python -m inventory\_pkg# ```# # > Don't use `python ./inventory\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\_pkg` — relative imports will break.# # \---# # \## Input Files# # Dump your files in the same folder as `inventory\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\_pkg/`. It detects them automatically by filename:# # | Pattern | Source |# |---|---|# | `MASS\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\*.zip` / `MASS\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\*.xlsx` | Shopee |# | `price\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\*.xlsx` / `\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\*export\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\*.xlsx` | Lazada |# | `\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\*SellerStockTemplate\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\*.xlsx` | Zalora |# | `EXL\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\*.xls` / `EXP\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\*.xls` / `EXP\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\*.xlsx` | Ordazzle |# | `Article\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\*.xlsx` | SAP |# | `modify\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\_\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\*.xlsx` | Manual override |# # Needs at least one channel file + Ordazzle + SAP or it will exit early.# # \---# # \## GUI — Generate For# # When you run the app, the \*\*Generate For\*\* section lists every detected brand/marketplace combination as a checkbox. Select which ones you want to generate output for.# # \- \*\*Zalora rows\*\* — a \*\*Brand:\*\* dropdown appears inline. Pick the matching brand from your `credentials.py` so the app knows which API key to use. Selecting a brand automatically fetches `ZAL\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\_MPSKU\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\_ID` and `ZAL\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\_MPITEM\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\_ID`.# \- \*\*Modify rows\*\* — a \*\*Zalora:\*\* dropdown appears inline. Set to \*\*None\*\* if this file isn't a Zalora job, or pick a brand if you want Zalora MPID extraction alongside the modify output.# # \---# # \## Brands# # | Group | Brands | Ordazzle Warehouse | SAP Site |# |---|---|---|---|# | SSIEBG | Springfield, Cortefiel, MakeRoom \& More, DKNY, Dune London, Pazzion, Macarena, Nine West, Polo Ralph Lauren, Pomelo, Lacoste, women'secret, Lush, Calvin Klein, Tommy Hilfiger, Clarks, Superga | `SSIEBG\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\_WH\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\_EBGWarehouse` | `2W06` |# | PAYLESS | Payless, Payless ShoeSource | `Payless\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\_WH\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\_Warehouse` | `30W9` |# | SLCI | Old Navy, Gap, Banana Republic | `SLCI\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\_WH\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\_Warehouse` | `36W2` |# # If a brand isn't recognised it reads everything without filtering and prints a warning.# # \---# # \## Output# # Files are saved to `inventory\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\_pkg/RESULT/`:# # ```# Inventory\_Result\_Lacoste\_Zalora\_2026-05-05\_16-39.xlsx# ```# # \### Colour coding# # | Colour | Meaning |# |---|---|# | Green | OK — values match |# | Red | Mismatch — needs attention |# | Yellow | Warning |# | Orange | Check this |# # \---# # \## Hardcoding Files Manually# # If auto-detect isn't picking up your files, edit the top of `main.py`:# # ```python# CHANNEL\_FILES  = \[r"C:\\data\\MASS\_Lacoste.zip"]# ORDAZZLE\_FILES = \[r"C:\\data\\EXP\_export.xls"]# SAP\_FILES      = \[r"C:\\data\\Articles.xlsx"]# ```# # Set back to `None` to return to auto-detect.# # \---# # \## SKU Format# # Valid SKUs are 10–13 digits starting with `1` or `2`. Anything else is skipped.# # \---# # \## File Structure# # ```# inventory\_pkg/# ├── credentials.py           ← your API keys (gitignored)# ├── credentials.example.py   ← safe template (committed)# ├── .gitignore# ├── README.MD# ├── \_\_main\_\_.py              ← entry point# ├── main.py                  ← main run() logic# ├── constants.py             ← brand names, warehouse codes, Excel styles# ├── file\_finder.py           ← auto-detects input files# ├── utils.py# ├── zalora\_api.py            ← Zalora API client# ├── gui/# │   ├── app.py               ← GUI window# │   └── zalora\_options.py    ← Zalora brand list# ├── channels/# │   ├── lazada.py# │   ├── shopee.py# │   └── zalora.py# ├── readers/# │   ├── ordazzle.py# │   ├── sap.py# │   └── modify.py# ├── output/# │   └── builder.py           ← builds final Excel output# └── RESULT/                  ← output files land here (gitignored)# ```
